# 简历分销与众包服务平台 - 系统设计与开发文档 (v1.0)

## 一、 项目概述
本项目基于开源仓库 `hotyue/ResumeCollection`，旨在将其丰富的简历模板资源转化为一个具备自动交付、多级分销与众包制作功能的商业化微电商平台。应用需完美适配抖音、小红书等短视频平台的内置浏览器，通过轻量级 H5 页面完成引流与变现。

## 二、 技术栈架构
- **前端 (H5)**: Vue 3 + Vite + Vant UI
- **后端 (API)**: Python (FastAPI)
- **数据库**: PostgreSQL
- **支付网关**: PayJS
- **部署与运行**: Docker & Docker Compose
